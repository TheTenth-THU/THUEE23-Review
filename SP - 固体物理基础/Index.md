## 课程内容

+ [[金属电子论初步]]：Drude 经典电子论 → Wiedemann–Franz 定律 → Sommerfeld 自由电子模型 → 费米球与费米面 → Sommerfeld 展开
+ [[固体的结构与结合]]：晶格几何 → 倒易点阵 → 化学键与结合
  + [[晶格结构]]：晶格几何 → Bravais 点阵 → 堆积比与配位数 → 晶列/晶向 → Miller 指数
  + [[倒易点阵]]：倒格子 → 倒格矢与晶面 → X 射线衍射 → 消光条件
  + [[固体的结合]]：分子轨道理论 → 化学键 → 轨道杂化 → 结合能与相互作用势
+ [[固体能带理论]]：Bloch 定理 → 近自由电子近似 → 紧束缚近似 → 准经典运动与 Bloch 振荡 → 三种布里渊区图景 → 直接/间接带隙与光吸收 → 有效质量
+ [[固体的电特性]]：从金属 Boltzmann 输运到半导体载流子统计、漂移-扩散与霍尔效应，再到接触现象的完整电学图景
  + [[半导体[[载流子输运过程]]与 Boltzmann 方程 → 载流子统计 → 漂移与迁移率 → 扩散与复合 → 非平衡载流子与准费米能级 → 霍尔效应
  + [[固体间接触的电特性]]：功函数与接触势 → 异质结 → pn 结平衡/非平衡/I-V/电容/击穿 → Schottky 结与热电子发射 → 欧姆接触 → MIS 结构
+ [[固体的热特性]]：晶格振动 → 声子 → 爱因斯坦模型 → 德拜声子态密度与 $T^{3}$ 定律 → 热传导与热膨胀
+ [[固体的磁特性]]：原子磁矩 → 磁化率 → 抗磁/顺磁/铁磁/反铁磁/亚铁磁性 → 超导
