### 知识点导航

操作系统是管理计算机硬件与软件资源的**系统软件**，向下管理硬件、向上服务应用，是计算机系统中最基本、最核心的软件基石。课程围绕操作系统的**五大管理功能**展开：

+ **[[操作系统引论]]** — OS 的定义与多视角理解、发展历史、基本特征（并发/共享/虚拟/异步）、主要功能（处理机/存储/设备/文件/接口管理）、用户接口（命令接口/程序接口）、设计原则与体系结构
+ **[[处理机管理]]** — 处理机管理是整个 OS 核心，涵盖进程抽象、线程模型、进程间通信和处理机调度与死锁四大子主题，构成一条完整的逻辑链：进程定义并发执行的基本单位 → 线程分离资源分配与调度执行 → 进程间通信提供并发体协调机制 → 调度与死锁解决 CPU 分配策略与资源竞争僵局
  + **[[进程]]** — 进程的描述：程序与并发执行、进程定义与 PCB 结构、进程状态模型（三/五/七态）与转换、进程控制原语
  + **[[线程]]** — 线程引入动机、进程与线程对比、用户级/内核级线程、混合实现、多线程模型
  + **[[进程间通信]]** — 临界区与互斥、同步机制（软件/硬件/信号量/管程/消息传递）、经典同步问题（生产者-消费者/读者-写者/哲学家进餐/理发师）、高级 IPC（共享内存/消息传递/管道）
  + **[[处理机调度]]** — 调度层次与时机、抢先/非抢先调度、调度算法（FCFS/SJF/HRRN/RR/优先级/MLFQ/实时调度）、死锁四条件与处理策略（预防/避免（银行家算法）/检测/解除）
+ **[[存储器管理]]** — 存储体系、连续分配（单一/分区）、离散分配（页式/段式/段页式）、虚拟存储、页面置换算法(OPT/FIFO/LRU/Clock)
+ **[[设备管理]]** — I/O 硬件原理、I/O 控制方式（程序控制/中断/DMA）、I/O 软件层次、缓冲与 SPOOLing、磁盘调度(FCFS/SSTF/SCAN)、RAID
+ **[[文件管理]]** — 文件与目录概念、逻辑/物理组织、存储空间分配、空闲空间管理、文件共享与保护、可靠性（备份/一致性检查）
